# Valorant Tracker Web Platform

전문적인 발로란트 전적 검색 및 분석 웹 플랫폼입니다.

## 🚀 프로젝트 개요
React와 Zustand를 활용한 프론트엔드, FastAPI를 활용한 백엔드로 구성된 확장 가능하고 유지보수가 용이한 아키텍처를 기반으로 합니다. 
Docker, Nginx, AWS EC2를 이용해 배포되며, 우수한 보안, 환경 관리 및 반응형 UI/UX 디자인을 제공하는 것을 목표로 합니다.

## 🛠 기술 스택
- **프론트엔드**: React, Zustand (상태 관리)
- **백엔드**: FastAPI (Python)
- **인프라 및 배포**: Docker, Nginx, AWS EC2
